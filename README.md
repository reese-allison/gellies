# Gellies

Animated gelly web buddies! Customize your own Gelly and watch it hop around a 3D room.

**Live demo:** https://reese-allison.github.io/gellies

## Quick Start

```bash
git clone https://github.com/reese-allison/gellies.git
cd gellies/frontend
npm install
npm start
```

Open http://localhost:3000

## Build for Production

```bash
npm run build
```

Output will be in `frontend/dist/`

## Tech Stack

- Preact
- MUI (Material-UI) v5
- GSAP animations
- Webpack
